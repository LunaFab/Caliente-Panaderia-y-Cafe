Análisis del Sistema

Análisis Estructurado

Modelado detallado del sistema de gestión para la cafetería OCHETO utilizando metodologías de análisis estructurado. Incluye diagramas de contexto, flujo de datos y entidad-relación.

Modelo Ambiental

Diagrama de Contexto

Representación de alto nivel que muestra el sistema como un único proceso y sus interacciones con entidades externas.

A) Declaración de Propósitos

El Sistema de Ventas para la Cafetería "Ocheto" es una solución de software integral diseñada para automatizar y optimizar los procesos operativos y administrativos del negocio. Su propósito principal es reemplazar los métodos manuales actuales por un sistema digital que permita:

  • Gestión eficiente de ventas y pedidos
  • Control en tiempo real del inventario
  • Generación de reportes analíticos
  • Mejora en la experiencia del cliente

B) Diagrama de Contexto

Diagrama de contexto que muestra las interacciones entre el sistema y las entidades externas

C) Lista de Acontecimientos

N° Acontecimiento:
1. Cajero ingresa datos para registrar una nueva venta.
2. Cajero ingresa datos para consultar la disponibilidad de un producto.
3. Cajero ingresa datos para modificar un pedido pendiente.
4. Cajero ingresa datos para cancelar un pedido antes de su finalización.
5. Gerente ingresa datos para dar de alta a un nuevo producto en el menú.
6. Gerente ingresa datos para dar de baja a un producto del menú.
7. Gerente ingresa datos para dar de alta a un nuevo usuario del sistema (cajero/administrador).
8. Gerente ingresa datos para dar de baja a un usuario del sistema.
9. Gerente ingresa datos para registrar la entrada de nueva mercadería al inventario.
10. Gerente solicita datos para generar un reporte de ventas.
11. Gerente solicita datos para generar un reporte de inventario.
12. Gerente solicita datos para generar un reporte de rendimiento del personal.
